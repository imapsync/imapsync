Imapsync list of imap server softwares supported (and the failures one) (back to menu)

To know weither your IMAP server is a widespread choice, take a look at http://openemailsurvey.org/.

Let's start with the long reported success stories list: 72 different imap software servers supported!
[host1] means "source server" and [host2] means "destination server":

Please report to the author (gilles.lamiral@laposte.net) any success or bad story with imapsync and, if you know them, mention the IMAP server software names and version on both sides. This will help future users. You can grab these values, software name and release number, by looking at two lines at the beginning of the output. Example: 

 ...
 Host1 banner:* OK louloutte Cyrus IMAP4 v1.5.19 server ready
 Host2 banner:* OK Courier-IMAP ready

You can use option --justconnect to get those lines. Examples:

imapsync --host1 test1.lamiral.info \
         --host2 test2.lamiral.info \
         --justconnect
