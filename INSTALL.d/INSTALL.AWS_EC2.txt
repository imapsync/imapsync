=======================================================================


AWS: Amazon Web Service

A) cat /etc/system-release
Amazon Linux release 2 (Karoo)

  sudo yum install -y https://dl.fedoraproject.org/pub/epel/epel-release-latest-7.noarch.rpm
  sudo yum install imapsync
